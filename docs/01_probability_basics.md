# ðŸŽ² Probability Basics

---

## 1. Probability of an Event

\[
P(A) = \frac{\text{Favorable Outcomes}}{\text{Total Outcomes}}
\]

**Use:** Estimate risk of an event, e.g. a market correction.

---

## 2. Expected Value

\[
E(X) = \sum P(x) \cdot x
\]

**Use:** Expected return over time for a random variable like price.

---

## 3. Variance and Standard Deviation

\[
\text{Var}(X) = \sum P(x) \cdot (x - E(X))^2
\]
\[
\text{Std Dev} = \sqrt{\text{Var}(X)}
\]

**Use:** Measures volatility in returns or outcomes.

---

## 4. Covariance and Correlation

\[
\text{Cov}(X, Y) = \sum P(x, y)(x - E(X))(y - E(Y))
\]
\[
\text{Correlation } \rho\_{X,Y} = \frac{\text{Cov}(X, Y)}{\sigma_X \cdot \sigma_Y}
\]

**Use:** Portfolio diversification, asset comparison.
